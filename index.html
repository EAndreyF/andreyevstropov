Евстропов Андрей — JavaScript разработчик

Я специализируюсь на разработке клиентской части веб приложений, так называемый front-end developer. В случае необходимости могу разработать и серверную часть.

Сайты, выходящие из под моего пера обладают:
  1. Интерактивностью
  2. Адаптивностью под различные устройства
  3. Высокой производительностью
Профессионально владею:
  1. HTML5
  2. CSS3
  3. Чистым JavaScript (BOM, DOM, ECMAScript 5)
  4. Библиотеками jQuery, jQuery UI
  5. MVC фреймоворком Backbone
Для тестирования использую:
  1. QUnit
  2. Jasmine
Шаблонизация на основе:
  1. XSLT
  2. Handlebars
  3. Underscore

Обожаю общаться с бекендом по RESTful JSON API

При разработке проповедую методологию TDD и стараюсь внедрять лучшие практики методологий XP и Scrum, в зависимости к чему готова команда.

Обеспечение модульности: Require.js

Системы контроля версий на основе Git: GitHub, Bitbucket. Работа из командной строки.

Серверную часть разрабатываю на PHP. Использую фреймворк CakePHP.

Базы данных:
  1. MySQL - в основном
  2. Oracle
  3. MS SQL
Багтрекеры:
  1. JIRA
  2. Bugzilla
  3. Redmine

В качестве скриптов автоматизации по работе с баг трекерами использую Python.

В процессе разработки так же сталкивался с библиотеками Angular.js и Node.js
